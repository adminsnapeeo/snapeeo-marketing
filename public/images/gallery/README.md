# Gallery strip photos

Name your files in this folder:

```
snapeeo-gallery-1.webp
snapeeo-gallery-2.webp
snapeeo-gallery-3.webp
…
```

Then set `GALLERY_STRIP_COUNT` in `src/config/images.ts` to match how many photos you've added.

If you use `.jpg` or `.png` instead of `.webp`, change `GALLERY_STRIP_EXT` in the same file (e.g. `'.jpg'`).

Only these files appear in the home page gallery strip.
